# Исходники

Основной исходный файл:

- `checker_go.go` - полный исходный код Go-версии чекера.

Готовый файл для запуска:

- `checker_go.exe` - собранная Windows-версия, Go для запуска не нужен.

README специально описывает только запуск готового exe, потому что этот репозиторий рассчитан на обычных пользователей.
